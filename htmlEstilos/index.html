  1. Estilo em linha

    2. Diz-se que uma folha de estilo está em linha ou inline quando as regras de estilo estão declaradas dentro da Tag HTML.

    Observe dentro da Tag p o estilo declarado na forma inline, em destaque:

    <p style="color: blue;" >Texto do parágrafo </p>

    Exercício

    Use estilo em linha para mudar a cor de um título de nível 1 - h1, para vermelho

  2. Estilo incorporada

    3. As folhas de estilo incorporadas são declaradas na Tag head para determinar regas de estilo para uma única página HTML.

    Observe dentro da Tag p o estilo declarado na forma incorporada, em destaque:

    <style type="text/css">
    p {margin-left: 20px;}
    </style>

    Exercício

    Use estilo incorporado para mudar a distância da margem esquerda de um título de nível 1 - h1, para 80px do lado esquerdo.

  3. Estilo externo

    4. Assim como as incorporadas, as folhas de estilo externas são chamadas na Tag head. Contudo, utiliza-se um arquivo com a extensão .css que é vinculado a página HTML por meio da Tag link.

    Observe dentro da Tag link o estilo declarado na forma externa para vincular o arquivo css à página HTML, em destaque:

    <link rel="stylesheet" href="./css/stilo.css">

    Exercício

    Use estilo externo para mudar a cor do fundo da página por meio do propriedade background no seletor html

  4. Estilo importado

    5. Essa forma de usar folhas de estilo é apropriada quando se quer alterar um estilo definido previamente.

    Declara-se entre as Tags head e em style com type="text/css",conforme o destaque abaixo:

    <style type="text/css">
    @import url(novo.css);
    </style>

    Exercício

    Use estilo incorporado para mudar a cor do fundo da página por meio do seletor background na tag html.

  5. Todas as formas em um único documento

    6. É possível usar as três formas de folha de estilo no mesmo documento. Contudo, deve-se observar o "efeito cascata" das folhas de estilo. O efeito cascata das folhas de estilo é um método para aplicação de uma regra css em função do pêso (importância e prioridade).

    Exercício

    1. Em um arquivo html utilize todas as 3 formas para dar estilo a página.
    2. Aplique as 3 formas de utilizar estilo ao mesmo elemento HTML com seletores iguais e valores diferentes e responda como os estilos são priorizados conforme a teoria do efeito cascata do CSS?
    3. Crie um menu de navegação para acessar as perguntas e suas respectivas respostas.
    4. Pesquise e reponda:
      1. Como determinar a ordem do efeito cascata?
      2. O que é folha de estilo alternativa?
      3. O que é folha de estilo persistente?
      4. O que é folha de estilho preferida?
